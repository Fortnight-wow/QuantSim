# QuantSim Architecture

## Main Components

### Market Engine
Generates and updates market prices.

### Candlestick Chart
Displays price movement using HTML Canvas.

### Portfolio Engine
Tracks cash, positions, and net worth.

### Trading Bots
Current bots:

- SMA Bot
- Pattern Bot
- Alpha 3 Bot
- Custom Bot

### Analytics
Calculates:

- Profit
- Sharpe Ratio
- Portfolio Performance
- Bot Comparison

## Future Architecture

Market Data
    ↓
Strategy Engine
    ↓
Risk Engine
    ↓
Execution Engine
    ↓
Portfolio Analytics
    ↓
Visualization Layer

## Long-Term Goal

Turn QuantSim into a browser-based quantitative research platform that can be used to learn financial engineering concepts and test trading strategies.
