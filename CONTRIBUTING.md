# Contributing to QuantSim

Thanks for checking out QuantSim.

This project is built like a student learning project, so simple improvements are very welcome.

## How you can help

- Fix bugs
- Improve chart quality
- Add new trading ideas
- Make the UI cleaner
- Improve documentation
- Add more bot strategies

## Before you edit

- Keep the main idea of the simulator intact
- Try not to remove existing features unless they are broken
- Keep code readable and easy to understand

## Suggested workflow

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Test the app in the browser
5. Open a pull request

## Style

- Keep code simple
- Use clear variable names
- Write comments only where they help understanding

## Custom bot ideas

Good custom bots can be based on:

- SMA or EMA crossovers
- RSI overbought/oversold logic
- Trend breakout logic
- Mean reversion
- Volatility filters

## Notes

This project is meant for learning, experimenting, and building a stronger quant portfolio.
