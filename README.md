# QuantSim

A quantitative trading simulator that I started building to learn financial engineering, algorithmic trading, and market microstructure through hands-on experimentation.

## Why I Built This

I am a student interested in:

- Financial Engineering
- Quantitative Finance
- Algorithmic Trading
- Mathematics
- Market Simulation
- Portfolio Analytics

Instead of only reading theory, I wanted a project where I could visualize strategies and see how trading decisions affect performance in real time.

## Current Features

- High quality candlestick chart
- Portfolio value tracking
- Manual trading system
- SMA trading bot
- Pattern recognition bot
- Alpha 3 trend-following bot
- Custom bot framework
- Bot performance comparison
- Sharpe ratio tracking
- Trade execution log
- AI-assisted analysis panels

## Custom Bot System

Users can create their own trading bot and compare it against the built-in bots.

Planned language support:

- JavaScript
- TypeScript
- Python
- Lua
- WebAssembly strategies
